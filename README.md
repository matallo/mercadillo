# Mercadillo

This repo contains the source code for [https://matallo.github.io/mercadillo](https://matallo.github.io/mercadillo).


## Installation

```
yarn # npm install to install with npm.
```


## Usage


### Run locally

```
yarn run start
```


### Build

```
yarn run build
python -m SimpleHTTPServer
```


### Lint

```
yarn run lint
```
